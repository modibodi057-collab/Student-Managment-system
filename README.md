# Student Management System (CLI)

A robust Java-based Command Line Interface (CLI) application designed to manage student records dynamically. This project demonstrates core backend development principles, object-oriented programming (OOP), and database integration using **SQLite** and **JDBC**.

---

## 🚀 Features

The system provides a fully functional interactive menu to perform complete **CRUD** operations:

* **Create (Option 1):** Add new students with specific attributes like Name, Age, and Grade.
* **Read (Option 2):** View a formatted list of all registered students fetched directly from the database.
* **Delete (Option 3):** Remove student records permanently using their unique Student ID.
* **Update (Option 4):** Modify existing student information seamlessly.
* **Search (Option 5):** Look up a specific student's details instantly.
* **Data Persistence:** Integrated with a local SQLite database file to ensure data is saved permanently even after closing the application.

---

## 🛠️ Tech Stack

* **Language:** Java (JDK 11+)
* **Database:** SQLite
* **Driver:** SQLite JDBC Driver
